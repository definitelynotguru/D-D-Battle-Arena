A browser-based, turn-based Dungeons & Dragons–inspired combat game built with HTML, Tailwind CSS, and vanilla JavaScript. Choose a hero class, roll dice, manage abilities and cooldowns, and defeat progressively stronger enemies in a classic D&D-style battle arena.

FEATURES:

Playable Classes: Fighter, Wizard, Rogue, Cleric — each with unique stats and abilities

Dice-Based Combat: d20 attack rolls, damage dice, critical hits, and armor class checks

Turn-Based System: Player and enemy turns with round tracking

Abilities & Cooldowns: Attacks, healing, buffs, and special moves

Multiple Enemies: Goblins, Orcs, Skeletons, and a Dragon boss

Combat Log: Detailed action history for every turn

Polished UI: Tailwind CSS, Font Awesome icons, animations, and visual feedback

Responsive Design: Playable on desktop and mobile browser
